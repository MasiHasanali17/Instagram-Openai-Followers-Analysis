## üìÑ What Data Was Used?

The dataset contains raw Instagram profile information collected from followers of OpenAI.
Each profile typically includes:

* Username
* Number of posts
* Number of followers
* Number of accounts they follow
* Profile name
* Category/type of page
* Bio/description

This raw text data was cleaned, structured, and analyzed inside a Jupyter Notebook.

---

## üîç How the Data Was Processed

1. The raw text file was split into individual profile sections.
2. Each section was parsed to extract important fields like posts, followers, etc.
3. Special notations such as **K** (thousand) and **M** (million) were converted into numerical values.
4. A structured dataset was created for easy analysis.
5. Simple comparisons were made to identify maximum values and category counts.
